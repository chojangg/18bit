## 게임방법
1. 스페이스바를 눌러 게임을 시작
2. 게임이 시작되고 목숨 3개가 주어진다
3. 방향키로 캐릭터를 좌우로 이동하여 피구공을 피한다
4. 목숨을 다하게 되면 게임이 종료된다
5. 이름을 입력하면 지금까지의 게임 기록을 확인할 수 있다

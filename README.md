# 18bit
하늘에서 떨어지는 피구공을 피하는 게임
<br>
## 게임방법
- 캐릭터를 방향키로 좌우 이동
- 피구공을 피하면 점수가 올라감
- 피구공에 3번 맞게되면 게임이 끝남

## 필요한 라이브러리, 데이터
 - pygame
 - time
 - sys
 - random
 - threading
 - os
 - DB
 - tkinter
